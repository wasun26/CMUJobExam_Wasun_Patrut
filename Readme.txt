การติดตั้ง MySQLi DB
1. ทำการติดตั้ง DB บน MySQLi Server จากไฟล์ cmu_room.sql
2. เริ่มใช้งานได้ตามปกติ
3. ไฟล์ config.onc.php คือ ที่เก็บ Setting การเชื่อมต่อ
